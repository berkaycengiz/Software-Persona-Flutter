# Mini Katalog Uygulamasi

## Kisa Aciklama
Bu proje, modern bir mobil arayuz tasarimina sahip Apple urunlerinin listelendigi ve sepete ekleme simülasyonunun yapildigi interaktif bir katalog uygulamasidir.

## Kullanilan Flutter Surumu
* Flutter: 3.41.9
* Dart: 3.11.5

## Calistirma Adimlari
Projeyi yerel ortamda calistirmak icin terminal uzerinden app dizinine gidin ve sirasiyla su komutlari uygulayin:

1. Bagimliliklari yukleyin:
   ```bash
   flutter pub get
   ```

2. Uygulamayi baslatin:
   ```bash
   flutter run
   ```
